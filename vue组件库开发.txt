安装脚手架
  npm i -g vue-cli

  vue init webpack <项目名称>

cd 项目
  
  安装依赖包
  npm i
  
启动项目：
   npm run dev

vux核心组件，布局组件-viewBox
   import {viewBox} from 'vux'
   import 'vux/src/style/reset/less'
   在官网找到viewBox查看如何使用
     使用viewBox要做一些基础样式
	   html,body{
	     margin:0;
		 width:100%;
		 height:100%;
		 overflow0x:hidden;
	    }
	   #app{
	     height:100%;
	   }
   
   安装npm i -D less-loader
   
   在style里引入less文件，要声明style标签里声明lang='less'
