vue笔记
脚手架工具
1.安装vue
npm install -g vue-cli
2.新建项目
vue init webpack myproject
   对项目的命名及配置初始化
   cd myproject
3.安装依赖npm install
4.跑npm run dev

方法：
method:function(){}
监听数据的变化要做什么事
watch:{
	a:function(val,oldVal){
      console.log(val.oldVal)
	}
}

vue下的一个机制：在vue对象里的方法都可以拿到data里的数据

数据渲染：
  v-text="a"  
  v-html="a"  以html的方式渲染
  {{a}}

模板指令：
  v-if="" 显示隐藏
  v-for="item in items" 循环
  v-on:click="doThis" 事件绑定,这个doThis是在method里取得
  v-bind:src=""
  v-bind:class=""
  这里的class可以通过该变量来控制是否出现,如以下写法
  {red:isRed}
  [classA,classB]
  [classA,{classB:isB,classC:isC]

v-on:keyup.enter="" 键盘敲击enter时进行的操作

如何划分组件
   使用es6里的一个写法把模板加载进来
   import Header from './header.vue'
   improt Footer from 'footer'
在页面上<header></header><footer></footer>通过vue渲染出来
   new Vue({
      data:{
      isShow:true
      },
      components:{
        Header,Footer
      }
   })
